# Role: Security & secrets scanner

## Responsibilities:
- Audit AES-GCM encryption flow in `src/lib/crypto.ts` for Zero-Knowledge BYOK architecture
- Verify Master Key derivation (PBKDF2/Argon2) and ensure API keys are never stored in plain text
- Verify that API keys are only decrypted in RAM during request execution and never persisted unencrypted
- Implement and audit Prompt Injection Guard logic (OWASP Top 10 for LLM Applications)
- Scan for hardcoded credentials, API keys, and tokens in all files
- Verify `.env` is in `.gitignore` and never committed
- Audit CORS configuration in `server/index.ts` — ensure `origin` is restricted for SaaS production
- Validate rate limiting headers and protection against brute-force/DoS attacks
- Check for XSS vectors in `dangerouslySetInnerHTML` usage (Shiki highlight layers in Editor.tsx)
- Audit `package.json` for known vulnerable dependencies (`npm audit`)
- Verify Zod-based validation of all LLM outputs before they reach sensitive sinks

# Product

## Register

product

## Users
Senior Developers, Security Auditors, and SREs who need precise, high-density code analysis without visual fluff.

## Product Purpose
Refract is a high-end code audit terminal. It provides "Industrial Grade" analysis, highlighting vulnerabilities and providing secure fixes through a "Brutalist-Cyber" interface that prioritizes precision and technical density.

## Brand Personality
Industrial, Precise, Raw, Technical. High-density information, terminal-first, zero-fluff.

## Anti-references
- Soft SaaS "cream" dashboards with rounded corners (>2px).
- Purple-to-blue "AI" gradients.
- Generic Lucide icons without technical backgrounds.
- Vague, prose-heavy summaries without technical grounding.

## Design Principles
1. **Terminal First**: Monospaced typography and micro-caps for all metadata.
2. **Pedagogical Friction**: High-impact actions require intentional UX friction (confirmation).
3. **Industrial Rigidity**: Strict grids, 90-degree corners, and sharp (2px or hairline) borders.
4. **Bimodal Density**: Contrast between massive macro-typography and tightly packed telemetry.
5. **Tactical Telemetry**: Use cold "Void" backgrounds with neon "Volt" accents for high-contrast visibility.

## Accessibility & Inclusion
- WCAG 2.1 AA contrast ratios (4.5:1 minimum).
- Monospaced fonts for clarity in code and data.
- Reduced motion support via Framer Motion's `useReducedMotion`.
